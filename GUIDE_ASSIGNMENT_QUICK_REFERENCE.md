# Guide Assignment - Quick Reference Card

## 🚀 Quick Steps

```
1. Open Guest List
2. Click "Assign Guide"
3. View Eligible Guides
4. Choose: Manual OR Auto-Assign
5. Confirm Details
6. ✅ Done!
```

## 🎯 What Gets Checked

| ✅ Must Pass | Impact |
|------------|--------|
| Approved guide | Required |
| No expired certs | Required |
| Available on date | Required |
| No time conflicts | Required |
| Not flagged | Required |

## ⚠️ Warnings (Can Still Assign)

| ⚠️ Warning | Action |
|-----------|--------|
| Cert expires soon | Acknowledge & proceed |
| Schedule overlap | Review & confirm |
| Multiple tours | Check workload |

## 📋 Assignment Status Codes

| Status | Means | Next Action |
|--------|-------|------------|
| 🟡 Pending | New, not confirmed | Click Confirm |
| 🟢 Confirmed | Locked in | Tour proceeds |
| 🔵 Completed | Tour finished | View only |
| ⚫ Cancelled | Removed | Need new guide |

## 🎮 Actions Available

**Pending Assignment:**
- ✅ Confirm → Make it official
- ❌ Cancel → Remove assignment

**Confirmed Assignment:**
- ✅ Complete → Mark as done
- ❌ Cancel → Remove if needed

## 💬 What to Include in Notes

```
✓ Group size & composition
✓ Special needs or requests
✓ Accessibility requirements
✓ Dietary restrictions
✓ Specific instructions
✓ Any concerns
```

## 🚫 When NOT to Assign

| Issue | Solution |
|-------|----------|
| ❌ Cert expired | Wait for renewal |
| ❌ Time conflict | Choose different guide |
| ❌ Unavailable | Check availability calendar |
| ❌ Flagged | Contact admin |
| ❌ No guides available | Reschedule tour date |

## 📱 Auto-Assign vs Manual

| Aspect | Auto-Assign | Manual |
|--------|-------------|--------|
| Speed | ⚡ Fast | Normal |
| Control | System picks | You choose |
| Best for | Quick assignments | Specific needs |
| Risk | Low (system scores) | Low (you verify) |

## 🏆 What Auto-Assign Considers

1. Fewer current assignments (-10 pts each)
2. More experience (+0.5 pts/year)
3. Fresh certifications (-15 if expiring)
4. Availability (+5 if free)

**Higher score = Better pick**

## 🔔 Guide Gets Notified About

- ✉️ Tour date & time
- 👥 Number of guests
- 📍 Tour type/service
- ⚠️ Any warnings
- 📝 Your special notes

## ❌ System Blocks (Hard Rules)

These **CANNOT** be overridden:
- ❌ Expired certifications
- ❌ Time conflicts
- ❌ Guide marked unavailable
- ❌ Recent compliance flags

## ✅ Best Practices

1. ⏰ Assign 3-5 days early
2. 📞 Contact guide to confirm
3. 📝 Document special needs
4. 🔍 Review all warnings
5. 📅 Check availability calendar
6. 💬 Communicate expectations
7. 📋 Keep records updated

## 🎯 Scoring Example

**Guide A:**
- 1 current tour → -10
- 5 years exp → +2.5
- Cert OK → 0
- Available → +5
= **97.5 points** ⭐

**Guide B:**
- 2 current tours → -20
- 8 years exp → +4
- Cert expires in 10 days → -15
- Available → +5
= **74 points**

→ **Guide A selected** ✅

## 🆘 Troubleshooting

| Problem | What to Do |
|---------|-----------|
| No guides available | Check certs, reschedule date, contact admin |
| Assignment fails | Refresh, try different guide |
| Guide didn't get notification | Call guide, check their email |
